# CDSS — Epilepsia Neonatal
Sistema de apoyo diagnóstico para síndromes epilépticos neonatales basado en ontología OWL/SWRL.

## Stack
- Frontend: Next.js + Tailwind + TypeScript → Vercel
- Backend: Flask + Python → Railway
- Motor de inferencia: SWRLAPI + Drools (API oficial Protégé)
